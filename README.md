# API Testing with Postman – GitHub Gists API

---

## Overview

This project demonstrates API testing using Postman, including authentication, CRUD operations, environment variables, and automation, as required by the assignment rubric.

---

## API & Authentication

* **API:** GitHub Gists API (REST v3)
* **Base URL:** [https://api.github.com](https://api.github.com)
* **Auth:** Personal Access Token (Bearer)

---

## CRUD Operations

POST `/gists` • GET `/gists/{id}` • PATCH `/gists/{id}` • DELETE `/gists/{id}`

---

## Postman & Automation

* Environment variables: `base_url`, `token`, `gist_id`
* Tests added for status codes and responses
* Collection run using Postman Collection Runner
